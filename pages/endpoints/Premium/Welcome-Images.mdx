## Welcome Image Generator

Much like üîó[Welcome Image Templates](/endpoints/Welcome-Images) this endpoint generates a templated welcome image.<br/>
The difference with this endpoint is there are no preset backgrounds.

You can use this endpoint via the `/premium` endpoint as such<br/>
<Code style={{ fontWeight: 400 }}><span style={{color: colors.requests["GET"]}}>GET</span> /premium/welcome/<span style={{color: colors.requests["PARAM"]}}>:template</span></Code>

The templates are the same as the templates show in the üîó[Welcome Image Template Documentation](/endpoints/Welcome-Images).<br/>
However here you can provide an image URL via the `bg` query parameter and that will be the background instead.
